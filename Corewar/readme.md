# 导入

> Core War（磁芯大战）是一款由两个或多个程序（代替它们的作者）玩的游戏，这些程序用一种名为*Redcode*的汇编语言编写而成，并在一个名为 _MARS（Memory Array Redcode Simulator，内存阵列 Redcode 模拟器）_ 的虚拟机中运行。游戏的目标是使对方程序的所有进程终止，从而使你的程序独占机器。

Core War 是一个有趣的游戏：相较于其它的编程游戏，这个*老牌游戏*并非是像它们那样**给机器人写一个脑子**，而是**真真正正地写一个机器人**，代码会实实在在地成为你的武器，这给它带来了独一无二的魅力与乐趣。

但由于有一定的入门门槛，且基本没多少中文文档，有也极为分散，CW 在国内基本是无人问津的状况。由此，笔者在此整合文档，并进行机翻+润色，以供入门与参考。

一篇文档如果有其 MD 文件，会将其拷贝至译后文档同一文件夹；否则会提供 TXT 版本（事实上，大多数文档也只有 TXT 版本，但翻译我会将其稍稍格式化为 MD ;-) ）
